How to Run&Install

unzip release1g6.zip and go into the folder with a terminal.

Subapplications

Registration

Name: registration.jar

This application is responsible for registrating racers during or after the race has commenced.

Run in the terminal with: java -jar registration.jar
Pick a file to save registrations and click the green button to register racers.
Start numbers can be added afterwards. If the start number is unreadable when the initial registration is ongoing simply press enter to register the driver without a startnumber.
If several racers are to begin simultaneously start times can be added later through the ResulMerge application. This needs to be enabled in the config file.

ResultMerge

Name: resultMerge.jar

Arguments:

-c "filename" specifies the name of the config-file. No specification of the file uses "config" as standard name.

This application is responsible for summarizing and sorting the results from the previous registration-generated files.

To change the settings of the merge the config-file can be edited for different settings, for example if the file is to be sorted, the specifics of the race and the settings to the path where the file will be created.

Examples

java -jar resultMerge.jar -o resultat.txt -s starttider.txt -f maltider.txt -n namnfil.txt -t maraton
java -jar resultMerge.jar -o resultat3.txt -m 00.01.37 -f maltider.txt -n namnfil.txt -t varvlopp
