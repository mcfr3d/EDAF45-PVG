Resultcalculator

Name: resultMerge arguments:
-o: output file path, mandatory
-n: name file path, mandatory
-f: finish file path, mandatory
-s: start file path, mandatory if not massstart
-t=maraton: [maraton, varvlopp], mandatory
-m: hh.mm.ss, massstart optional

Exempel

java -jar resultMerge.jar -o resultat.txt -s starttider.txt -f maltider.txt -n namnfil.txt -t maraton
java -jar resultMerge.jar -o resultat3.txt -m 00.01.37 -f maltider.txt -n namnfil.txt -t varvlopp
